01 — Concept

Where the Desert
Meets the Sea

Ananda Beach Camp sits at the edge of the Sinai desert, where terracotta mountains fall into turquoise water. The site needed backgrounds that felt alive — not stock photos with overlays, but hand-crafted generative scenes that breathe.

Each section of the homepage carries a different canvas effect, matched to its content. Light sections get warm, sandy textures. Dark sections get deep underwater scenes. Every effect responds to the mouse and respects prefers-reduced-motion.

Color Palette

Tech Decisions
  • Canvas 2D for procedural scenes (mountains, coral, water) — smaller bundle, wider support
  • WebGL + GLSL shaders for the tidal sand ripples — GPU-accelerated caustic math
  • simplex-noise for organic movement — Perlin-style noise without the grid artifacts
  • IntersectionObserver pauses off-screen canvases — zero wasted frames
  • ResizeObserver handles responsive and RTL layout shifts
  • getCanvasQuality() detects low-end devices — reduces DPR, particles, or skips entirely

03 — Process

From Prompt
to Production

Brik generates interactive HTML files from natural language. But shipping those files into a Next.js production site requires a deliberate conversion pipeline.

1

Prompt

Write a single-paragraph description specifying algorithms, color hex codes, interactivity, and performance constraints.

2

Generate

Brik outputs a self-contained HTML file with inline JS. Usually p5.js instance mode or vanilla Canvas 2D.

3

Convert

Strip p5.js dependency. Rewrite as a React useEffect with vanilla Canvas/WebGL. Add IntersectionObserver, ResizeObserver, and perf detection.

4

Ship

Integrate as a 'use client' component. Position absolute behind section content with pointer-events: none. Deploy to Vercel.

04 — Architecture

Performance
by Default

Five simultaneous canvas effects on a single page could destroy mobile performance. Every decision was made to keep the site at 60fps on mid-range devices.

Device Detection

getCanvasQuality() checks hardwareConcurrency, viewport width, and devicePixelRatio. Low-end devices get halved particle counts and DPR capped at 1.

Reduced Motion

If prefers-reduced-motion: reduce is set, canvases skip initialization entirely. Zero JS overhead, zero animation. The section backgrounds become solid colors via CSS overlays.

Off-Screen Pausing

IntersectionObserver sets an isVisible flag. The animation loop still runs (to avoid cold-start jank) but skips all draw calls when the canvas is scrolled out of view.

Offscreen Buffers

The Water Surface effect renders per-pixel noise at 0.35x resolution on an offscreen canvas, then upscales with bilinear smoothing. 8x fewer pixels calculated per frame.
